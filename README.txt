====================================================
  Screen Recorder for Windows — Quick Start Guide
====================================================

Thank you for downloading Screen Recorder!
This is a free, standalone Windows screen recorder.
No Python or FFmpeg installation required.

----------------------------------------------------
  HOW TO INSTALL
----------------------------------------------------

1. Double-click  ScreenRecorderSetup.exe
2. Follow the on-screen installer steps (Next → Next → Install)
3. Launch "Screen Recorder" from your Desktop or Start Menu

That's it — no extra software needed.

----------------------------------------------------
  HOW TO USE
----------------------------------------------------

BASIC RECORDING
  1. Open Screen Recorder
  2. Choose recording area:
       • Full Screen  — records your entire display
       • Custom Region — drag to select any part of the screen
  3. Choose audio source:
       • Microphone   — records your voice/mic
       • System Audio — records sounds playing on your PC
  4. Click  START  (or press F9)
  5. A 3-second countdown will appear, then recording begins
  6. Click  STOP  (or press F9 again) to finish
  7. Your video is saved as an MP4 file

PAUSE & RESUME
  • Press F10 (or click Pause button) to pause mid-recording
  • Press F10 again to resume — audio stays in sync automatically

MIC MUTE
  • Click the Mic toggle during recording to mute/unmute your mic
    without stopping the recording

OUTPUT RESOLUTION
  • Choose Native (your actual screen size), 1080p, or 720p
    from the Settings before starting

HOTKEYS
  F9  →  Start / Stop recording
  F10 →  Pause / Resume

----------------------------------------------------
  OUTPUT FILES
----------------------------------------------------

Recordings are saved as  .mp4  files (video + audio merged).
Default save location: your Videos folder
You can change the save location in Settings.

----------------------------------------------------
  SYSTEM REQUIREMENTS
----------------------------------------------------

  • Windows 10 or Windows 11 (64-bit)
  • ~200 MB free disk space (for the app)
  • Additional space for your recordings

----------------------------------------------------
  TROUBLESHOOTING
----------------------------------------------------

No audio recorded?
  → Make sure the correct audio device is selected
  → For system audio, enable "Stereo Mix" in Windows Sound settings:
     Right-click speaker icon → Sounds → Recording tab →
     Right-click empty area → Show Disabled Devices →
     Enable Stereo Mix

Video saves as .avi instead of .mp4?
  → This means FFmpeg was not found. Re-install using the
     official installer (ScreenRecorderSetup.exe) which
     bundles FFmpeg automatically.

App won't start?
  → Try running as Administrator (right-click → Run as administrator)
